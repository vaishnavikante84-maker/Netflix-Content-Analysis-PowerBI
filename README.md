# Netflix Content Analysis Dashboard | Power BI

This project presents a detailed Power BI dashboard analyzing the **Netflix Movies & TV Shows Dataset**.  
The dashboard provides insights into content distribution, release trends, genre popularity, and country-wise contribution to the Netflix catalog.


 Dashboard Highlights

 1. Movies vs TV Shows  
A comparison showing the proportion of movies and TV shows available on Netflix.

 2. Top 10 Countries by Number of Titles  
Identifies the countries contributing the highest number of titles to the Netflix platform.

 3. Release Year Trend  
A line chart visualizing how Netflix content has increased over the decades.

 4. Genre Distribution  
A treemap showing popular genres/categories of Netflix titles.

 5. Rating Slicer  
Interactive filter allowing users to explore titles based on TV ratings (e.g., TV-MA, PG-13, R).

 Tools & Skills Used

- **Power BI Desktop**
- Data Cleaning & Preprocessing (Power Query)
- Data Visualization & Dashboard Design
- Filters, Slicers, and DAX basics
- Exploratory Data Analysis (EDA)
